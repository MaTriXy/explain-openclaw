#### Canvas Agent Actions Restricted to Trusted Surfaces ([`8e04d1fe1`](https://github.com/openclaw/openclaw/commit/8e04d1fe1))

The macOS canvas A2UI action message handler now restricts agent action dispatch to trusted in-app canvas schemes only (`CanvasScheme.allSchemes`). The previous `isLocalNetworkCanvasURL()` check that allowed local-network HTTP pages to dispatch agent actions was removed. This prevents arbitrary web content from triggering agent execution through the canvas handler, tightening the trust boundary for which surfaces can invoke canvas agent actions.

#### Tlon Allowlist Enforcement Hardened ([`3cbf93241`](https://github.com/openclaw/openclaw/commit/3cbf93241))

Tlon channel allowlist validation now checks `!== undefined` instead of `.length > 0`, ensuring explicit empty allowlists (`[]`) fail closed rather than falling back to config defaults. Additionally, `resolveAllCites()` is deferred until after channel authorization checks, preventing untrusted senders from triggering cite expansion before permission validation completes.

#### External Content Metadata Sanitized ([`a97b9014a`](https://github.com/openclaw/openclaw/commit/a97b9014a))

Adds `sanitizeMetadataValue()` helper inside `wrapExternalContent()` that applies `replaceMarkers()` and strips newlines (`/[\r\n]+/g` → space) to the `sender` and `subject` metadata fields. This prevents injection of marker sequences or multiline content into metadata headers of wrapped external content boundaries.

#### Strict Mode Tools Disabled for Non-Native APIs ([`9616d1e8b`](https://github.com/openclaw/openclaw/commit/9616d1e8b))

Adds `supportsStrictMode` flag to `normalizeModelCompat()` in `model-compat.ts`. For non-native OpenAI-compatible APIs, `supportsStrictMode` defaults to `false`, preventing OpenAI's `strict: true` tool validation from being sent to providers that silently ignore or reject it. Without this fix, tool calls could be silently dropped by incompatible API endpoints.

#### Compaction Timeout Hardening ([`6a458ef29`](https://github.com/openclaw/openclaw/commit/6a458ef29))

Wraps the `opts?.onCancel?.()` callback invocation in `compactWithSafetyTimeout()` with a try-catch block. This prevents exceptions thrown during compaction cancellation from interrupting the timeout/abort control flow, maintaining safety timeout integrity even if the cancellation hook throws.

#### Android CallLog Handler with Parameterized SQL ([`d7ac16788`](https://github.com/openclaw/openclaw/commit/d7ac16788))

Introduces `CallLogHandler.kt` for Android with runtime permission checking (`ContextCompat.checkSelfPermission` for `READ_CALL_LOG`), parameterized SQL queries (`LIKE ?` placeholders), and safe input coercion (`toIntOrNull()`, `coerceIn(1, 200)` for limit, `coerceAtLeast(0)` for offset). Adds `CALL_LOG_COMMANDS` to `node-command-policy.ts` to gate the new command through the node command policy.
