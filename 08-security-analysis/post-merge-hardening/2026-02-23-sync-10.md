### Session Lock Hardening ([`1becebe18`](https://github.com/openclaw/openclaw/commit/1becebe18))

Session lock acquisition hardened against race conditions in contended scenarios. Added `LockInspectionDetails` type alias to encapsulate lock inspection metadata, `lockInspectionNeedsMtimeStaleFallback()` to validate staleness detection conditions, and `shouldReclaimContendedLockFile()` implementing mtime-based fallback for stale lock detection when PID/createdAt metadata is unavailable or corrupted. Improved resource cleanup in `acquireSessionWriteLock()` with explicit handle close and file removal in error path to prevent dangling file handles. Captured `nowMs` once and reused across lock inspection calls to prevent time inconsistencies and TOCTOU windows.
