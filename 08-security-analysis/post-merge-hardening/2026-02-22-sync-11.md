## Key Security Changes

### Symlink Escape Blocking ([`f4dd0577b`](https://github.com/openclaw/openclaw/commit/f4dd0577b))

Hook transform path resolution now validates symlinks using `realpath.native()` and checks that resolved paths remain within expected ancestor directories. This blocks path traversal attacks through symlink chains in hook configuration.

### Channel Token Hardening ([`ae8d4a8ee`](https://github.com/openclaw/openclaw/commit/ae8d4a8ee))

Replaced `Math.random()` with CSPRNG (`crypto.randomBytes()`) for Slack external menu tokens and Urbit channel ID generation. Centralized secure random generation in new infrastructure helpers with token validation patterns.

### Exec Allowlist Wrapper Resolution ([`2b63592be`](https://github.com/openclaw/openclaw/commit/2b63592be))

Added transparent unwrapping of environment variable wrappers and shell-dispatch wrappers during exec allowlist analysis. This prevents bypasses where malicious payloads are smuggled through wrapper commands that appear superficially safe.

### HOME/ZDOTDIR Env Blocking ([`c2c7114ed`](https://github.com/openclaw/openclaw/commit/c2c7114ed))

Extended host environment sanitization to block `HOME` and `ZDOTDIR` environment variable overrides. This prevents attackers from pointing shell startup files (`~/.zshrc`, `~/.bashrc`) to attacker-controlled locations before allowlist-evaluated commands execute.

### ACP Gateway Startup Sequencing ([`9f0b6a8c9`](https://github.com/openclaw/openclaw/commit/9f0b6a8c9), [`7499e0f61`](https://github.com/openclaw/openclaw/commit/7499e0f61))

Implemented comprehensive gateway startup validation that waits for hello message confirmation before processing ACP messages. Added explicit failure handling when sandbox host is configured but unavailable, ensuring fail-closed behavior.

### Information Exposure Reduction ([`2c6dd8471`](https://github.com/openclaw/openclaw/commit/2c6dd8471))

Removed `host` and `commit` fields from hello-ok frames in WebSocket protocol, reducing passive information leakage during connection establishment.
